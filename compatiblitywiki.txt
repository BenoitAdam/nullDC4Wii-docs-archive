== Retest broken games please ! ==


<b>Chuchu Rocket is launchable since alpha 0.26 !! Please re-test all broken games that haven't been tested at least on alpha0.26 !</b>

== Testing procedure & creating the best preset ==

Test on Dolphin or a Real Wii. Dolphin render can be better than Wii

Report any differences between those two

Creating the best preset for the game method :

1/ Launch with default preset

2/ play a little bit the game (default stage, just input A/Start/A), 2/3 minutes is enough

3/ If some things doesn't display good, try to change settings. Already we know that works :

* fine (seam) lines in 2D sprites : put "LOW" as graphics
* Z fighting : put FIXED DEPTH to "tight"
* 2 player viewport (or 2 cameras in the same scene) = SPLITSCREEN

4/ Flickering, fix can be :

* BLEND_MODE (set to off)
* TRANS_MODE (set to off)
* ASYNC_RENDER (set to on)
* DEPTH-CLIP (set to tight)

5/ Other graphical things not display try :

* 4BPP/8BPP to CI4_FAST or CI8_FAST
* BG POLYGON to ON
* PUNCH TROUGH to ON
* TRANS_SORT to ON
* RENDER TO TEX to ON
* 2D Famebuffer = ON

6/ off Canvas problem (screen too small or too big)

* try xratio = on (or off)
* CANVAS_WIDTH (try different values)

7/ Restart the game, try CACHE_VERY_FAST and CACHE_FAST. Find the lowest compatible option for the game. They may crash heavy-scene thus, like Shenmue intro

8/ If you want to dig more, try the game in early version of NullDC4Wii :

* alpha0.25 = no sound and few compatibility but might get some good suprise
* alpha0.28 = the state after
* alpha 0.40 = good performance

If you find a regression, find the exact version before/after the regression

Always use CACHE_QUALITY to see it it fixes your graphical glitch.

If you need advice, describe your graphical problem in Claude or Gemini and attach the gxrend.cpp file of the project and ask him what's your problem and what preset to try


Check here for example screenshot : 
https://github.com/BenoitAdam/nullDC4Wii/issues/86

== WinCE games ==

WinCE games aren't supported (It demands extra power... maybe Wii U can achieve some result, but probably not Wii)

You can still add them and put [WinCE] on it for information

List of all WinCE games here : https://segaretro.org/Category:Dreamcast_Windows_CE_games
