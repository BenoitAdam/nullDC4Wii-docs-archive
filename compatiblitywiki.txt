== Retest broken games please ! ==


<b>Chuchu Rocket is launchable since alpha 0.26 !! Please re-test all broken games !</b>

=== Note 2 : Use I4STUB and I8STUB parameter if experiencing very low fps (around 1 FPS) ===

== Testing procedure ==

Test on Dolphin or a Real Wii

Dolphin render can be better than Wii

Report any differences

== WinCE games ==

WinCE games aren't supported (It demands extra power... maybe Wii U can achieve some result, but probably not Wii)

You can still add them and put [WinCE] on it for information

List of all WinCE games here : https://segaretro.org/Category:Dreamcast_Windows_CE_games
